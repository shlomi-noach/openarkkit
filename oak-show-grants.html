oak-show-grants

NAME

oak-show-grants: show accounts privileges in GRANT format

SYNOPSIS

Show grants for all accounts:
oak-show-grants
Show grants for the 'webuser' user only:
oak-show-grants --account-user=webuser
Show grants for users on the localhost only:
oak-show-grants --account-host=localhost
Show grants for the 'webuser'@'192.168.0.120' account:
oak-show-grants --account-user=webuser --account-host=192.168.0.120
Show grants for the 'webuser' user only, provide connection parameters, prompt for password:
oak-show-grants --user=root --ask-pass --socket=/tmp/mysql.sock --account-user=webuser
Same as above, use defaults file:
oak-show-grants --defaults-file=/home/myuser/.my-oak.cnf --account-user=webuser

DESCRIPTION

oak-show-grants scans the mysql.users table to find all accounts, then lists the grants for each account. The grants are presented in a GRANT format, which can later be executed on another machine. It is possible to filter out certain users, certain hosts or specific accounts.

OPTIONS

--account-user=ACCOUNT_USER

A specific user for whom to show grants

--account-host=ACCOUNT_HOST

A specific host for which to show grants

--ask-pass

Prompt for password.

--defaults-file=DEFAULTS_FILE

Read from MySQL configuration file. Overrides --user, --password, --socket, --port.

Configuration needs to be in the following format:

[client]
user=my_user
password=my_pass
socket=/tmp/mysql.sock
port=3306

-H HOST, --host=HOST

MySQL host (default: localhost)

-p PASSWORD, --password=PASSWORD

MySQL password

-P PORT, --port=PORT

TCP/IP port (default: 3306)

-S SOCKET, --socket=SOCKET

MySQL socket file. Only applies when host is localhost

-u USER, --user=USER

MySQL user

ENVIRONMENT

Requires MySQL 4.1 or newer, python 2.3 or newer. python-mysqldb must be installed in order to use this tool. You can
apt-get install python-mysqldb
or
yum install mysql-python
